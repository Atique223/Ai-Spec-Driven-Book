# Physical AI & Humanoid Robotics Book

This repository contains a comprehensive educational resource on Physical AI and Humanoid Robotics, built with Docusaurus.

## Overview

This book provides a comprehensive guide to:
- Physical AI concepts and embodied intelligence
- ROS 2 fundamentals (middleware, nodes, topics, services)
- Digital twin concepts with Gazebo and Unity
- Humanoid robotics and simulation environments

## Features

- 10+ chapters organized by modules
- Practical examples and exercises
- Integration with simulation tools (Gazebo, Unity)
- Assessment materials with knowledge checks
